# Coco Color 微信小程序

这是网站的微信小程序原生版本（第一阶段：首页展示）。

## 怎么预览（5 步）

1. 下载安装「微信开发者工具」（稳定版）：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 打开工具 → 新建/导入项目 → 目录选择这个 `miniprogram` 文件夹。
3. AppID：
   - 想快速预览：选「测试号」或「不使用云服务 / 体验 AppID」即可在模拟器里看到效果。
   - 正式上线：填你企业主体小程序的 AppID（在微信公众平台 → 开发管理 → 开发设置里查看）。
4. 工具左侧「模拟器」即可看到小程序首页。
5. 想真机预览：点工具右上角「预览」，用你的微信扫码。

## 已包含内容

- 首页：品牌介绍、诊断老师、诊断项目（含报价）、诊断明细、预约流程、门店地址
- 门店地址支持「查看地图」（内置地图）和「复制地址」
- 内容数据在 `pages/index/index.js` 顶部的 `content` 对象里，改文字/价格直接改这里

## 还没做（后续阶段）

- AICOCO 答疑（需要后端，建议用微信云开发或现有 API）
- 线上诊断、评论区

## 上线前你需要做的

- 用企业主体登录微信公众平台，拿到 AppID 填进开发者工具
- 在「上传」后到公众平台提交审核、发布
- 门店经纬度目前用的是天河区近似值，正式上线可在 `index.js` 的 `contact.latitude/longitude` 改成门店精确坐标
