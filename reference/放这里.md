# cococolor 品牌素材文件夹

把素材放进对应子文件夹即可，文件名随意（最好用英文或拼音，避免太长）。

- `logo/` —— 品牌 logo（png/svg/jpg 都行，最好有透明底的 png）
- `teachers/` —— 诊断老师照片（一位老师一张或多张）
- `projects/` —— 诊断项目相关图片或文案（可选）
- `misc/` —— 其他参考图、配色截图、想模仿的网站截图等（可选）

## 怎么放进来

1. 打开「文件资源管理器」，进入：
   `桌面 → cococoloraiproject → reference`
2. 把你的 logo 文件直接拖进 `logo` 文件夹，老师照片拖进 `teachers` 文件夹。
3. 放好后回来告诉我「放好了」，我会读取并用在官网上。
